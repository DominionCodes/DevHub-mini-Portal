# Mini Devhub

A beginner-friendly React application that demonstrates modern frontend fundamentals including routing, API integration, reusable components, and dynamic pages.
The Portal allows users to browse developer profiles, view detailed information, and multi-page environment.

# Problem Statement
This project demonstrates how a real-world multi-page app works using:
- Dynamic routing
- API data fetching
  State management
  Component architecture

## ✨ Features
# Home Page
   Welcome section
   Interactive counter (useState + event handling)
# Users Page
   Fetch users from a public API
   Loading state while fetching
   Error handling
   Reusable UserCard components
# Dynamic User Profile Page
   Route: /users/:id
   Fetch individual user data
   Display detailed information (email, phone, company)
   Back navigation support
# About Page
 Static informational page
    
## 🧠Concepts Covered
This project intentionally covers all foundational React topics:
    Components
    Props
    Event Handling
    React Hooks (useState, useEffect)
    Routing (React Router)
    Dynamic Routing (useParams)
    API Fetching (Fetch API)
    Conditional Rendering
    List Rendering with map()
    
## 🛠 Tech Stack
  Frontend
  React.js
  React Router DOM
